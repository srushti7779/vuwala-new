========================================
IMPORT IN 3 SIMPLE STEPS
========================================

STEP 1: Create Database
------------------------
mysql -u root -p

CREATE DATABASE your_db_name CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
EXIT;


STEP 2: Import SQL File
------------------------
mysql -u root -p your_db_name < complete_test_database.sql


STEP 3: Update Yii2 Config
---------------------------
Edit: config/db.php

return [
    'dsn' => 'mysql:host=localhost;dbname=your_db_name',
    'username' => 'your_username',
    'password' => 'your_password',
];

========================================
TROUBLESHOOTING
========================================

ERROR: Foreign key constraint fails
FIX: The SQL file already has SET FOREIGN_KEY_CHECKS=0 at the start

ERROR: Access denied  
FIX: Check MySQL username/password, grant permissions:
     GRANT ALL ON your_db_name.* TO 'your_user'@'localhost';

ERROR: Character encoding issues
FIX: Import with UTF-8:
     mysql --default-character-set=utf8mb4 -u root -p your_db_name < complete_test_database.sql
